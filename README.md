# Sistem Penghitubf Kredit Sederhana Berbasis Web

Program sederhana untuk menghitung angsuran kredit mobil berdasarkan harga mobil (OTR), down payment (DP), dan jangka waktu kredit.

## Deskripsi
Aplikasi ini dibuat menggunakan **HTML dan JavaScript** untuk mensimulasikan perhitungan angsuran kredit mobil.  
Perhitungan bunga dan angsuran mengikuti alur **flowchart kredit mobil**.

## Fitur
- Input harga mobil (OTR)
- Input persentase DP
- Input jangka waktu kredit (bulan)
- Penentuan bunga otomatis berdasarkan jangka waktu
- Menampilkan angsuran per bulan

## Aturan Bunga
- ≤ 12 bulan : 12%
- > 12 dan ≤ 24 bulan : 14%
- > 24 bulan : 16,5%

## Cara Menjalankan
1. Download atau clone repository ini
2. Buka file `index.html`
3. Jalankan langsung di browser

## Teknologi
- HTML
- CSS
- JavaScript

## Catatan
Program ini tidak memerlukan XAMPP atau database karena seluruh proses dilakukan di sisi client.
